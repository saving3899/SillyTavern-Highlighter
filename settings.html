형광펜 설정

기본 설정

보관 모드는 채팅 삭제 후에도 형광펜을 유지합니다. 체크 해제 시 플로팅 버튼이 숨겨지며, 요술봉 메뉴에서만 접근할 수 있습니다. 체크 시 형광펜 모드가 항상 활성화되며, 비활성화할 수 없습니다.

색상 커스터마이징

형광펜 색상을 원하는 대로 변경할 수 있습니다. 각 색상 버튼의 색상, 불투명도, 글자색을 설정하세요.

저장 방식

📁 현재 저장 위치

형광펜은 SillyTavern의 설정 파일(settings.json)에 자동 저장됩니다.

💾 백업 방법

독서노트 패널의 "백업" 버튼을 눌러 JSON 파일로 다운로드할 수 있습니다.

사용 방법

1. 형광펜 모드 활성화

플로팅 버튼 클릭 후, 패널의 형광펜 아이콘 클릭

2. 텍스트 형광펜 칠하기

채팅 메시지에서 텍스트 드래그 → 색상 선택

3. 메모 추가

형광펜 클릭 → 메모 입력

4. 패널 이동

패널 헤더를 드래그하여 이동 (데스크톱)
